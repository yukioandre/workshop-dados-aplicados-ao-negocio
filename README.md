# Datasets das Lives: Dados Aplicados ao Negócio

Este repositório reúne os datasets utilizados nas lives do workshop **Dados Aplicados ao Negócio**, realizadas nos dias **01/06 e 02/06**.

O objetivo destes arquivos é apoiar os exemplos práticos apresentados nas aulas, mostrando como a análise de dados pode ser usada para entender problemas reais de negócio, acompanhar indicadores, gerar insights e apoiar a tomada de decisão.

Ao longo das lives, os dados serão usados para explorar temas como:

- análise de desempenho do negócio;
- leitura de indicadores;
- identificação de problemas e oportunidades;
- construção de raciocínio analítico;
- conexão entre dados, métricas e decisões estratégicas.

## Sobre o workshop

O workshop **Dados Aplicados ao Negócio** foi criado para profissionais e estudantes da área de dados que querem ir além da parte técnica.

A proposta é mostrar que saber usar ferramentas como **Excel, SQL, Power BI, Python ou IA** é importante, mas não suficiente. Para gerar impacto de verdade, o profissional de dados precisa entender o contexto do negócio, fazer boas perguntas e transformar análises em recomendações úteis.

## Estrutura do repositório

Os arquivos deste repositório podem incluir bases em formatos como `.csv`, `.xlsx` ou similares, dependendo dos exemplos usados durante as lives.

Cada dataset será utilizado com fins educacionais e pode ser adaptado para estudos, exercícios, análises exploratórias e criação de dashboards.

## Como usar

Você pode baixar os arquivos deste repositório e utilizá-los para acompanhar as lives, reproduzir as análises apresentadas ou praticar por conta própria depois do workshop.

A recomendação é que você tente não apenas executar os passos mostrados, mas também explorar novas perguntas de negócio a partir dos dados.

## Observação

Os datasets disponibilizados aqui têm finalidade didática. Eles podem ter sido simplificados, adaptados ou simulados para facilitar o aprendizado e destacar os principais conceitos trabalhados nas lives.
